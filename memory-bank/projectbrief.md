# Project Brief

- Project: Temp Inbox Manager
- Scope: Desktop tool for temporary email and phone inbox workflows.
- Current stack target: Tauri + SvelteKit + TypeScript.
- Key integrations: Mail.tm APIs and imported SMS URL pools.
- Primary constraints: local persistence, polling-based refresh, no cloud sync in V1.
- [LESSON] Windows paths containing `&` require PowerShell npm script-shell to avoid command parsing failures.
